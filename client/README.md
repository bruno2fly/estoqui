# Estoqui — React app (Phase 1)

## Run

```bash
cd client
npm install
npm run dev
```

Open http://localhost:5173. You should see the sidebar and placeholder pages for Dashboard, Inventory, Catalog, Vendors, History, and Settings.

## Build

```bash
npm run build
npm run preview   # preview production build
```
